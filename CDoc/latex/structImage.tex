Structure représentant l\textquotesingle{}image source pixelisée. 

Les pixels sont stockés dans un tableau 1D unique pour optimiser le cache. L\textquotesingle{}accès se fait via la formule \+: y \texorpdfstring{$\ast$}{*} W + x.
